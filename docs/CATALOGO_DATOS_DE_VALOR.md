# CATÁLOGO DE DATOS DE VALOR
## Espacio de Datos Sectorial FENITEL

---

## INTRODUCCIÓN

Este documento identifica los tipos de datos que aportarían valor al Espacio de Datos FENITEL, organizados por categoría sectorial. Estos datos, compartidos de forma agregada y anonimizada cuando corresponda, permitirían a los asociados obtener inteligencia de mercado, realizar benchmarking y tomar mejores decisiones empresariales.

---

## 8. FORMATO Y ESTRUCTURA DE DATASETS

### Formato Recomendado

Todos los datasets deben subirse preferentemente en formato:
- **CSV** (separador punto y coma, codificación UTF-8)
- **JSON** (estructura documentada)

### Estructura Mínima Recomendada

Cada dataset debe incluir:

| Campo | Descripción | Obligatorio |
|-------|-------------|-------------|
| `fecha_datos` | Fecha o período de los datos | Sí |
| `ambito_geografico` | Provincia, comunidad o nacional | Sí |
| `categoria` | UTP, ICT, FM, SAT | Sí |
| `subcategoria` | Según clasificación anterior | No |
| `descripcion` | Explicación del contenido | Sí |
| `unidad_medida` | Euros, unidades, metros, etc. | Sí |
| `fuente` | Origen de los datos | Sí |
| `metodologia` | Cómo se han recopilado | No |

### Ejemplo de Estructura CSV

```csv
fecha_datos;ambito_geografico;categoria;tipo_dato;valor;unidad_medida;fuente
2025-Q4;Nacional;UTP;metros_cat6a_instalados;125000;metros;Registro_interno
2025-Q4;Madrid;ICT;proyectos_vivienda_nueva;45;unidades;Certificaciones_OCA
2025-Q4;Cataluña;SAT;instalaciones_vsat;12;unidades;Base_clientes
```

## 9. PRIVACIDAD Y ANONIMIZACIÓN

### Principios de Compartición

1. **Agregación:** Los datos individuales se agregan para evitar identificación
2. **Anonimización:** Se eliminan referencias directas a clientes o proyectos específicos
3. **Temporalidad:** Datos con antigüedad mínima de un trimestre
4. **Consentimiento:** Solo se comparten datos autorizados por cada empresa
5. **Proporcionalidad:** Se comparte solo lo necesario para el objetivo

### Datos que NO se comparten

- Nombres de clientes
- Direcciones exactas de instalaciones
- Precios de proyectos individuales
- Información financiera detallada de empresas
- Datos personales de empleados

## 10. VALOR ESPERADO

### Para el Asociado Individual

- **Benchmarking:** Comparar su rendimiento con la media del sector
- **Inteligencia de mercado:** Conocer tendencias y demanda
- **Precios de referencia:** Ajustar presupuestos competitivamente
- **Oportunidades:** Identificar nichos y colaboraciones

### Para FENITEL

- **Datos sectoriales:** Argumentario para negociaciones y posicionamiento
- **Estudios de mercado:** Base para informes y publicaciones
- **Defensa del sector:** Datos para interlocución con administraciones
- **Valor añadido:** Servicio diferencial para asociados

### Para el Sector

- **Profesionalización:** Estándares y mejores prácticas compartidas
- **Visibilidad:** Datos que demuestran el valor del sector
- **Colaboración:** Red de empresas con capacidades complementarias
- **Innovación:** Identificación temprana de tendencias
